2019-12-05-温控器接口文档

Posted on 2020-06-24

上品尚生温控器接口文档

报文格式

表1 报文格式
Domain Description Length(bytes)
Magic num 固定值0xAA 1
Length 报文长度,发送的所有字节相加，最大长度72 1
Version 协议版本号，固定值0x10 1
Msgtype 消息类型（详细见表2） 1
DeviceType 设备类型 1
DeviceIndex 设备索引,固定值00 1
PacketID 报文序列号。由请求设备产生，响应和请求中Packet ID应相同 1
Type1 属性类型 1
Data1 属性值 4
... ... ...
... ... ...
Typen 属性类型 1
Datan 属性值 4

注：写数据的时候发送Type+Data，读数据的时候只发送Type

消息类型

表2 消息类型
Msgtyp Description 发送方
0x00 要求设备注册 网关
0x01 设备注册响应 设备
0x02 要求设备离网 网关
0x03 设备操作 网关
0x04 设备操作结果 设备
0x05 设备状态查询 网关
0x06 设备状态上报 设备
0x07 设备定位 设备

以上关于设备的消息类型应全部实现

DeviceType Description
05 地暖

属性类型和属性值定义

Type类型和Data数据
Type Value Data Value
响应结果 0x00 操作成功 00
操作失败 01
非法请求 02
参数错误 03
系统忙 04
故障 0xFF ERROR_SOCKET 01
ERROR_PURIFIER 02
设备工作状态 0x03 STANDBY 0x02
设备工作 0x01
停止工作 0x00
设备设定温度 0x09 温度取值范围 0~100
环境温度数据 0x40 取值范围 带1位小数，数据除以10为当前数据，单位摄氏度。如收到数据0x0115，数值为227，当前温度即为22.7度

设备定位

  • 地暖设备定位
Magic num Length Version Msgtype DeviceType DeviceIndex PacketID
AA 07 10 07 05 00 00

设备需要实现定位功能，定位功能需要用特定方式主动上报，比如按键

设备入网

设备离网

  • 设备入网离网一般由设备发起（按键），发送AT+JOIN申请加入网络，设备收到要求注册报文或者状态查询报文后将状态修改为已经入网状态。
  • 设备再15秒内未收到任何消息则将网络状态设置为离网状态。
  • 设备主动离网发送AT+LEAVE命令。
  • 设备屏幕上要显示设备的网络状态

消息举例

设备注册

  • 请求消息：要求地暖注册，网关->设备
Magic num Length Version Msgtype DeviceType DeviceIndex PacketID
AA 07 10 00 FF 00 ED
  • 响应消息：地暖注册响应，设备->网关
Magic num Length Version Msgtype DeviceType DeviceIndex PacketID
AA 07 10 01 05 00 ED

环境温度读取

  • 请求消息：向地暖请求温度数据，网关->设备
Magic num Length Version Msgtype DeviceType DeviceIndex PacketID Type
AA 08 10 05 05 00 ED 40
  • 响应消息：地暖返回数据22.7℃，设备->网关
Magic num Length Version Msgtype DeviceType DeviceIndex PacketID Type Data
AA 0C 10 06 05 00 ED 40 00000115

全部状态读取

  • 请求消息：广播读取设备的全部信息
Magic num Length Version Msgtype DeviceType DeviceIndex PacketID
AA 07 10 05 FF 00 ED
  • 响应消息：设备->网关，返回地暖设备的全部信息（设备工作状态、空调系统主机模式、温控器工作模式、风速、环境温度数据）
Magic num Length Version Msg type Device Type Device Index Packet ID Type1 Data1
AA 2A 10 06 05 00 ED 03 00 00 00 01
Type2 Data2 Type3 Data3 Type4 Data4 Type5 Data5
04 00 00 00 00 05 00 00 00 00 06 00 00 00 02 40 00 00 01 15

设备状态上报附加规则

  • 手动修改设备（温控器）的参数需要及时上报到网关，消息类型为0x06,PacketID为0x00，
    比如手动修改温控器的模式需要及时主动上报
  • 每两个以Magic num开头的数据包之间最少间隔50ms
  • 水阀状态改变要及时上报到网关
